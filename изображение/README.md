# Папка изображений для карточек игроков

Эта папка содержит все изображения, используемые для генерации карточек игроков.

## Структура

```
изображение/
├── icons/          # Иконки характеристик
│   ├── strength.svg       # Иконка силы
│   ├── endurance.svg      # Иконка выносливости
│   ├── charisma.svg       # Иконка харизмы
│   ├── agility.svg        # Иконка ловкости
│   └── intelligence.svg    # Иконка интеллекта
└── logo/           # Логотип приложения
    └── logo.png           # Логотип GOPRIME
```

## Использование

Все пути к изображениям настроены в `card-generator.js` и используют относительные пути от корня проекта.

### Фото пользователей

Фото пользователей хранятся в папке `player_photos/` и загружаются динамически при создании карточки.

## Примечания

- Все SVG иконки должны быть в формате SVG
- Логотип должен быть в формате PNG
- Размеры логотипа: 251.5x223px (автоматически масштабируется в коде)
